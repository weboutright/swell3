What is Swell?

Swell is a self-hosted booking system that runs entirely in your Google account. Unlike other booking platforms, you own your data, pay no monthly fees, and get unlimited bookings.

100% Free

No monthly fees, no booking limits, no hidden charges. Ever.

Your Data

Everything stored in YOUR Google account. You own it all.

5 Min Setup

Copy one file, deploy, configure. Start taking bookings today.

How It Works

You copy one file (Code.gs) to Google Apps Script, deploy it, and get your own booking system. It includes an admin dashboard to manage services, a booking widget to embed on your website, and automatic Google Calendar integration.

What You'll Need

1

Google Account

Free personal or business Google account. This is where your booking system will run.

2

Website or Blog

Any website where you want to embed the booking widget (WordPress, Wix, custom site, etc.)

3

Payment Processor Account (Optional)

Stripe or Square account if you want to accept payments. Can skip this if you're offering free services.

Setup Instructions

1

Copy Code.gs File

Download or copy the Code.gs file from the Swell repository.

Get Code.gs from GitHub
2

Create Apps Script Project

  1. Go to script.google.com
  2. Click "New project"
  3. Delete the default code in the editor
  4. Paste the entire Code.gs file you copied
  5. Name your project (e.g., "My Booking System")
  6. Click the save icon (üíæ)

Tip: Make sure you paste the ENTIRE file - it should be around 900+ lines of code.

3

Deploy as Web App

  1. Click "Deploy" button (top right)
  2. Select "New deployment"
  3. Click the gear icon ‚öôÔ∏è next to "Select type"
  4. Choose "Web app"
  5. Configure these settings:
    • Description: "Booking System v1"
    • Execute as: User accessing the web app ‚ö†Ô∏è (CRITICAL!)
    • Who has access: Anyone
  6. Click "Deploy"
  7. Copy your deployment URL (it will look like: https://script.google.com/macros/s/AKfyc.../exec)

Important: "Execute as: User accessing the web app" is REQUIRED for data isolation. This ensures each user's data is stored separately.

4

Configure Your Admin Dashboard

  1. Open your deployment URL and add ?page=admin to the end
  2. Example: https://script.google.com/macros/s/AKfyc.../exec?page=admin
  3. You'll be redirected to login with Google - authorize the app
  4. You'll see your admin dashboard!

Configure These Settings:

  • Calendar ID: Enter "primary" to use your main Google Calendar
  • Timezone: Select your timezone
  • Services: Add your services (name, duration, price)
  • Business Hours: Set when you're available
  • Payment Processor: Add your Stripe or Square payment link (optional)

Embed Your Booking Widget

Once configured, you can embed your booking widget on any website using a simple iframe.

Get Your Embed Code

  1. In your admin dashboard, click "Deployment & Embed" in the sidebar
  2. Find the "iframe Embed" section
  3. Click "Copy Code"
  4. Paste it into your website's HTML

Example Code

<iframe src="https://script.google.com/macros/s/YOUR_ID/exec?page=widget" width="100%" height="600px" frameborder="0" ></iframe>

Frequently Asked Questions

Is Swell really free?

Yes! 100% free. No monthly fees, no booking limits, no hidden charges. You only pay standard payment processor fees (Stripe/Square) if you choose to accept payments.

Where is my data stored?

Everything is stored in YOUR Google account using Google's USER_PROPERTIES storage. You own all your data. We can't access it, and you can delete it anytime.

Do I need coding skills?

No! Just copy and paste. If you can follow these instructions, you can set up Swell. No coding knowledge required.

Can I customize the widget appearance?

Yes! In the admin dashboard, go to "Deployment & Embed" and customize colors, width, height, and theme. More customization options coming soon.

How do I accept payments?

Connect your Stripe or Square account in the admin dashboard. When customers book, they'll be redirected to your payment link to complete payment before confirming the booking.

What happens when someone books?

The booking is automatically added to your Google Calendar, and confirmation emails are sent to both you and your customer. You can view all bookings in the admin dashboard.

Can I have multiple services?

Yes! Add as many services as you want. Each can have its own name, duration, price, and meeting link. Your customers will see all services in the booking widget.

How do I update my settings?

Just visit your admin dashboard (YOUR_URL?page=admin) anytime. Changes take effect immediately.
